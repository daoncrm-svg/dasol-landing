---
description: 클로드 코드 스타일 미션 수행 (계획 -> 구현 -> 자동 테스트 -> 검증)
---

이 워크플로우는 새로운 기능을 개발할 때 에이전트가 독립적으로 기획부터 검증까지 수행하는 "클로드 코드(Claude Code)" 스타일의 흐름입니다.

1. 요구사항 분석 및 계획 수립
사용자의 요구사항을 분석하고 `implementation_plan.md`에 개발 계획, 파일 변경 내역, 예상 리스크를 작성합니다.

2. 컴포넌트 및 엔드포인트 생성 (병렬 처리)
// parallel
2-1. [백엔드] FastAPI 라우터 및 서비스, Pydantic 스키마를 생성합니다.
// parallel
2-2. [프론트엔드] Next.js 서버/클라이언트 컴포넌트를 분리하여 화면을 구성합니다.

3. 단위 테스트 작성 및 실행
기능 구현이 완료되면 프론트엔드 및 백엔드 테스트 코드를 작성합니다.
// turbo
4. 테스트 러너 실행 (`pytest` 또는 `npm test`)하여 에러 발생 시 자동 수정 루프를 가동합니다.

5. 최종 검증
// run workflow: verify-ui
프론트엔드인 경우 브라우저 렌더링 확인을 진행하고 `walkthrough.md`에 스크린샷이나 동작 결과를 기록합니다.
